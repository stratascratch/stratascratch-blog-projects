# StrataScratch Blog Projects

Companion code and datasets for articles published on the [StrataScratch blog](https://www.stratascratch.com/blog/).

## Structure

Each folder contains:
- `README.md` — Project overview and link to the published article
- `datasets/` — Data files used in the project
- `notebook.ipynb` — Code walkthrough

## Projects

| Project | Article |
|---------|---------|
| `classifier-tuning-experiment/` | We Tuned 4 Classifiers on the Same Dataset: None Improved |
| `missing-data-imputation/` | We Tried 5 Missing Data Imputation Methods: The Simplest Method Won (Sort Of) |

## Getting Started

1. Clone the repository
2. Navigate to the project folder
3. Follow the README instructions

## Requirements

Most projects use Python 3.9+ with common data science libraries. Check individual project READMEs for specific dependencies.
